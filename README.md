# Jam-Digital-dengan-Tampilan-Emoji-pada-Dot-Matrix-Berbasis-FPGA
Jam digital berbasis FPGA Cyclone IV menggunakan Verilog. Menampilkan waktu MM.SS pada seven segment display 4 digit dengan teknik multiplexing, serta menampilkan 6 emoji berbeda pada dot matrix 8x8 yang berganti otomatis setiap 10 detik. Dilengkapi dengan kontrol start, stop, dan reset melalui push button.
